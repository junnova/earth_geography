# Earth Culture 任务清单

> **最后更新：2026-04-03**
> **当前进度：阶段一~三已完成，阶段四部分完成，下一步从 T4.3 开始**

## 阶段一：项目搭建与地球渲染 ✅

### T1.1 初始化项目结构
- **描述**：搭建 Vue 3 + Vite + Three.js 前端项目，搭建 Node.js + Express 后端项目
- **产出**：`client/` 和 `server/` 两个可运行的空项目
- **依赖**：无
- **状态**：✅ 已完成

### T1.2 搭建基础 Three.js 场景
- **描述**：创建场景、相机（PerspectiveCamera）、渲染器（WebGLRenderer）、灯光（AmbientLight + DirectionalLight）、星空背景（Points）
- **产出**：打开页面可见黑色星空背景 + 灯光就绪
- **依赖**：T1.1
- **状态**：✅ 已完成

### T1.3 创建地球球体 + 手绘风格纹理
- **描述**：创建 SphereGeometry + MeshStandardMaterial，加载 NASA Blue Marble 真实底图 + Canvas 后处理为水彩风格（色调调整+边缘描边+纸质噪点），叠加拓扑图显示国境线，添加大气层光晕（ShaderMaterial）
- **产出**：页面展示手绘风格地球 + 大气层光晕
- **依赖**：T1.2
- **状态**：✅ 已完成

### T1.4 实现拖拽旋转 + 滚轮缩放
- **描述**：集成 OrbitControls，配置 enableDamping（惯性阻尼）、minDistance/maxDistance（缩放范围限制）、rotateSpeed，禁用平移
- **产出**：鼠标拖拽可旋转地球，滚轮缩放，松手后有惯性衰减
- **依赖**：T1.3
- **状态**：✅ 已完成

### T1.5 实现 Idle 自动缓慢自转
- **描述**：检测用户交互状态，Idle 3 秒后地球自动缓慢自转，用户操作时立即停止自转
- **产出**：无操作时地球缓慢旋转，操作时停止
- **依赖**：T1.4
- **状态**：✅ 已完成

## 阶段二：图标系统 ✅

### T2.1 经纬度转 3D 坐标工具函数
- **描述**：实现 `latLonToVector3(lat, lon, radius)` 工具函数，将经纬度转换为 Three.js 球面坐标
- **产出**：`three/utils.js` 中导出工具函数
- **依赖**：T1.1
- **状态**：✅ 已完成

### T2.2 国旗图标渲染
- **描述**：使用 PlaneGeometry + MeshBasicMaterial 实现贴在地球表面的国旗图标，lookAt 法线方向，flagcdn.com 获取 320px 高清国旗，Canvas 绘制圆角边框+阴影
- **产出**：可在指定经纬度位置渲染贴面国旗图标
- **依赖**：T2.1, T1.3
- **状态**：✅ 已完成（从 Sprite 改为 PlaneGeometry 贴面）

### T2.3 加载初始国家 Mock 数据
- **描述**：创建 40 个国家的 Mock 数据（国名、国家代码、经纬度、景点名、描述、Wikimedia 实图 URL）
- **产出**：地球上显示 40 个国家国旗图标
- **依赖**：T2.2
- **状态**：✅ 已完成（40国，含景点实图URL）

### T2.4 图标点击检测
- **描述**：使用 THREE.Raycaster 实现鼠标点击射线检测，区分点击与拖拽（允许少量移动），命中 Mesh 时触发事件
- **产出**：点击图标可触发回调，传递对应国家信息
- **依赖**：T2.3
- **状态**：✅ 已完成

### T2.5 地球背面图标遮挡处理
- **描述**：每帧计算每个 Marker 法向量与相机方向的点积，背面图标隐藏+透明度设为 0，边缘渐隐过渡
- **产出**：地球背面图标自然隐藏，旋转时平滑过渡
- **依赖**：T2.3
- **状态**：✅ 已完成
